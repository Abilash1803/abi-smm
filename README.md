# Social Boost

Modern social media marketing platform built with React + Vite + Tailwind CSS.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

## Stack

- React 18
- Vite
- Tailwind CSS
- React Router

## Structure

```
frontend/src/
├── components/    # UI components
├── pages/         # Route pages
├── utils/         # Utilities & API
└── config/        # Constants
```

## Scripts

```bash
npm run dev        # Development
npm run build      # Production build
npm run preview    # Preview build
```

## Environment

Create `frontend/.env`:

```env
VITE_API_URL=http://localhost:5000/api
```
