# Backend API

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
PORT=5000
MONGO_URI=mongodb://localhost:27017/socialboost
JWT_SECRET=your-secret-key
NODE_ENV=development
```

3. Start server:
```bash
npm run dev
```

Server runs on: http://localhost:5000
